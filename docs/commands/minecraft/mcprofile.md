*Parent: [minecraft](/commands/minecraft)*

Get a user's "minecraft profile".

*Syntax: `minecraft mcprofile [user]`*

# Permissions

Everyone can run this command.

# Arguments

## user

The user to get the profile of. Defaults to the user calling the command.

Type: [User Argument](/glossary/argument#user-arguments)

# Examples

An example usage of the command is `%minecraft mcprofile`

# Subcommands

The command has no subcommands.
